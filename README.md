# SF6対策メモアプリ

## 概要

ストリートファイター6のキャラクター対策を管理するためのメモアプリです。

対戦中や対戦後に気付いた内容をキャラクターごとに保存し、次回以降の対策に活用することを目的として作成しました。

## 使用技術

* HTML
* CSS
* JavaScript
* LocalStorage
* Git / GitHub

## 主な機能

### キャラクター別メモ管理

以下のキャラクターごとに個別の対策メモを保存できます。

* リュウ
* ケン
* 春麗
* 豪鬼

### メモ分類機能

キャラクターごとに以下の項目を管理できます。

* 注意技
* 確定反撃
* 立ち回り
* 自分用メモ

### 自動保存機能

入力内容はLocalStorageへ自動保存されます。

### 最終更新日時表示

各キャラクターの最終更新日時を確認できます。

### 検索機能

保存済みメモをキーワード検索できます。

## 制作背景

ストリートファイター6をプレイする中で、キャラクター対策を効率よく管理したいと考え、本アプリを制作しました。

ゲームを楽しみながら学習できるテーマを選び、HTML・CSS・JavaScriptの学習を兼ねて開発しました。
